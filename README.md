# 機能
Nature Remo APIを使って，センサ情報や家電の状態変化をGoogleスプレッドシートに記録し，ツィッターでつぶやく

# 注意
- 家電はエアコンとライトのみ
- 以下のプロパティをScript Editorから設定する必要がある．
  - SPREADSHEET_ID （スプレッドシート）
  - CONSUMER_API_SECRET (Twitter) 
  - CONSUMER_API_KEY (Twitter) 
  - REMO_ACCESS_TOKEN (REMO)
